# Evidence Explanation

## Bitwarden

The screenshots show the Bitwarden website and GitHub repository. I examined Bitwarden because it is a modern password management solution used to improve password security. The website highlighted features such as secure password storage, password generation, and protection against data breaches. The GitHub repository showed that Bitwarden is open source and uses security-first principles including end-to-end encryption and zero-knowledge encryption. This demonstrates transparency and allows the security community to inspect and improve the software.

From this research, I learned that Bitwarden helps address one of the most common cybersecurity problems: weak and reused passwords. It provides a secure way to manage credentials while encouraging stronger authentication practices.

## Cloudflare Zero Trust

The screenshots show Cloudflare's Zero Trust platform and product offerings. Cloudflare promotes a Zero Trust approach, where users and devices must continuously verify their identity before accessing resources. The product pages showed services such as Zero Trust Network Access, Secure Web Gateway, DDoS protection, Web Application Firewalls, and API security.

From this research, I learned that modern cybersecurity no longer relies solely on perimeter security. Cloudflare's solution demonstrates how organizations can protect applications, networks, and users through identity-based access control and continuous verification.

## CrowdSec

The screenshots and brochure show CrowdSec's collaborative threat intelligence platform. CrowdSec collects threat intelligence from thousands of installations worldwide and shares information about malicious IP addresses across the community. The brochure explains how CrowdSec uses collective intelligence, behavior-based detection, and real-time threat sharing to proactively identify and block cyber threats.

From this research, I learned that modern threat intelligence systems can improve security by sharing information between organizations. CrowdSec's approach helps users identify and block malicious activity before attacks can cause significant damage.

## Reflection

This activity helped me understand that state-of-the-art cybersecurity solutions focus on different layers of protection. Bitwarden protects credentials, Cloudflare protects access and infrastructure through Zero Trust principles, and CrowdSec provides real-time threat intelligence. Together, these technologies demonstrate how modern cybersecurity combines authentication, access control, and threat intelligence to defend against evolving cyber threats.
