# B3 – Evidence Summary

This activity investigated three proactive security implementations that help organizations identify, prevent, and mitigate cybersecurity threats before they result in security incidents.

## Intrusion Detection Systems (IDS)

I reviewed an Intrusion Detection System (IDS), which continuously monitors network traffic and system activity for suspicious behaviour. IDS solutions help security teams identify potential attacks early by detecting unusual patterns, unauthorized access attempts, and malicious activity. This proactive monitoring allows organizations to respond to threats before significant damage occurs.

## Threat Intelligence Platforms

I reviewed a threat intelligence platform that provides information about emerging cyber threats, attack techniques, vulnerabilities, and threat actors. Threat intelligence enables organizations to stay informed about the current threat landscape and implement defensive measures before attacks occur. This proactive approach improves risk awareness and supports informed security decision-making.

## Automated Patch Management

I reviewed macOS automatic software update settings, which automatically check for, download, and install security updates provided by Apple. Automated patch management is a proactive security implementation because it ensures vulnerabilities are addressed as soon as security fixes become available.

By automatically applying updates, systems remain protected against known security threats and vulnerabilities. This reduces the amount of time a device is exposed to security risks and helps maintain a secure computing environment without requiring manual intervention.

## Reflection

This activity demonstrated that proactive security measures focus on preventing incidents before they occur. Intrusion Detection Systems provide visibility into suspicious activity, threat intelligence improves awareness of emerging risks, and automated patch management reduces exposure to known vulnerabilities. Together, these implementations strengthen an organization's overall cybersecurity posture and improve resilience against cyber threats.
