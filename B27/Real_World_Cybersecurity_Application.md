# B27 – Apply a Learned Concept to a Real-World Application

## Introduction

Cybersecurity concepts are most effective when they are applied in real-world environments. This activity involved applying the concept of Multi-Factor Authentication (MFA) to improve the security of a personal GitHub account.

---

# Cybersecurity Concept Applied

Multi-Factor Authentication (MFA)

---

# Objective

The objective of this activity was to strengthen account security by enabling Multi-Factor Authentication on a real online service.

---

# Real-World Environment

GitHub is a widely used software development and version control platform that stores source code, projects, and development resources. Because GitHub accounts may contain valuable code and personal information, protecting access to these accounts is important.

---

# Implementation

I enabled Multi-Factor Authentication (MFA) on my GitHub account.

The MFA setup process involved:

1. Accessing GitHub account security settings.
2. Navigating to the Multi-Factor Authentication configuration page.
3. Configuring an authentication method.
4. Verifying the setup process.
5. Confirming that MFA was successfully enabled.

After activation, account access required both a password and a second authentication factor, providing additional protection against unauthorized access.

---

# Security Benefits

Enabling MFA provides several security benefits:

* Reduces the risk of account compromise from stolen passwords.
* Provides an additional layer of authentication.
* Improves protection against phishing and credential theft.
* Supports the principle of defence in depth.
* Enhances overall account security.

---

# Outcome

The GitHub account was successfully protected using Multi-Factor Authentication. The implementation demonstrated how cybersecurity concepts taught in the unit can be applied to improve security in a real-world environment.

---

# Reflection

This activity demonstrated the practical value of Multi-Factor Authentication as a security control. By applying a concept learned during the unit to a real online account, I improved the security of my GitHub account and gained a better understanding of how layered security mechanisms help protect against cyber threats.
